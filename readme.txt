  2. CONTROLLER SUPPORT

UNTITLED GOOSE GAME plays well with mouse and keyboard, though it also officially
supports Xbox One, Xbox 360 and PlayStation 4 controllers. Nintendo Switch Pro Controllers
should also work if connected via Bluetooth.
Your operating system might have its own problems with particular types of controller,
however. For reference, refer to this list:
http://guavaman.com/projects/rewired/docs/SupportedControllers.html

  3. DISPLAY SETUP

Resolution and display settings can be accessed from the in-game menu.
It can also be adjusted at launch by holding down the SHIFT key when opening the game,
or by using the "-show-screen-selector" command line argument.

•••••••••••••••••••••••••••••••••••••••••••••••••••••••••••••

  4. EXHIBITION MODE

We've added a special mode to make the game better suited for public exhibition contexts.

When activated, this mode disables the Options and Quit buttons from the in-game pause menu,
and the Reset button will erase all progress and resets the game to the beginning (please
note that this could erase progress made in the normal play mode).

The game will also reset itself if left idle (no controller inputs) for three minutes.

This can be activated by creating a shortcut to the executable game file, and adding
"-exhibitionMode" to its target path as a command line argument.
